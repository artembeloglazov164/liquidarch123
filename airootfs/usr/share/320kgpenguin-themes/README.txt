ПОЛОЖИТЕ СЮДА ZIP ФАЙЛЫ С ТЕМАМИ MACOS

Необходимые файлы:
1. MacSonoma-kde.zip          - Plasma тема
2. Whitesur-gtk-theme.zip     - GTK тема  
3. Whitesur-icon-theme.zip    - Иконки
4. Whitesur-cursors.zip       - Курсоры
5. Lightly.zip                - Application Style
6. Albert.zip                 - Launcher (опционально)

После добавления файлов пересоберите ISO:
cd build
sudo bash build.sh

Темы будут автоматически установлены при загрузке Live ISO!
